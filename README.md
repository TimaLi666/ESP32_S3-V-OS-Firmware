# 🛡️ V-OS S3 - Cyber Security Multi-Tool

Портативный инструмент для аудита безопасности на базе **ESP32-S3**. Сочетает в себе физический интерфейс (OLED) и мощный Web-пульт.

## 🚀 Основные возможности
*   **Web Remote Control:** Управление через смартфон (Tabs, Logs, Database).
*   **BadUSB (HID):** Автоматическая выгрузка паролей Wi-Fi с ПК.
*   **Phishing Portal:** Captive Portal с перехватом URL и логинов.
*   **WiFi Monitor:** Живой сниффер пакетов и поиск скрытых SSID.
*   **RF Analyzer:** График загруженности каналов 1-13.
*   **Deauth/Spam:** Атаки на беспроводные сети и устройства iOS.

## 🔌 Схема подключения (Pinout)

| Компонент | Пин ESP32-S3 |
|-----------|--------------|
| OLED SDA  | GPIO 8       |
| OLED SCL  | GPIO 9       |
| Button UP | GPIO 10      |
| Button DWN| GPIO 11      |
| Button SET| GPIO 21      |
| Button BACK| GPIO 18     |
| RGB LED   | GPIO 48      |

## 🛠 Установка
1. Установите **Arduino IDE**.
2. Добавьте поддержку плат **ESP32** (версия 2.0.11+).
3. Установите библиотеки: `Adafruit SSD1306`, `Adafruit GFX`.
4. В настройках платы включите **USB CDC On Boot: Enabled**.
5. Прошейте устройство.
